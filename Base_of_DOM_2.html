Задача

Даны два абзаца, содержащие своим текстом какие-то числа, и кнопка. По нажатию на кнопку выведите на экран сумму хранящихся чисел.

10

20

Решение


      let p1 = document.querySelector('.p1').textContent;
      let p2 = document.querySelector('.p2').textContent;
      let button = document.querySelector('.button1');
      button.addEventListener('click', function () {
        alert(+p1 + +p2);
      });

Задача

Даны два абзаца и две кнопки. Пусть нажатие на первую кнопку выводит текст первого абзаца, а нажатие на вторую кнопку - текст второго абзаца.

Привет!

Пока!

Решение:


      let p3 = document.querySelector('.p3').textContent;
      let p4 = document.querySelector('.p4').textContent;
      let button2 = document.querySelector('.button2');
      let button3 = document.querySelector('.button3');
      button2.addEventListener('click', () => alert(p3));
      button3.addEventListener('click', () => alert(p4));

Задача

Дан абзац и две кнопки. По нажатию на первую кнопку запишите в абзац слово 'привет', а по нажатию на вторую кнопку - слово 'пока'.

Какой-то текст

Какой-то текст

Решение:


      let p5 = document.querySelector('.p5');
      let p6 = document.querySelector('.p6');
      let button4 = document.querySelector('.button4');
      let button5 = document.querySelector('.button5');
      button4.addEventListener('click', () => (p5.textContent = 'Привет'));
      button5.addEventListener('click', () => (p6.textContent = 'Пока'));

Какой-то изменяемый текст

Какой-то изменяемый текст

Задача

Дан абзац и кнопка. В абзаце записано какое-то число. По нажатию на кнопку возведите значение абзаца в квадрат и запишите его обратно.

3

Решение:


      let p10 = document.querySelector('.p10');
      let button6 = document.querySelector('.button6');
      button6.addEventListener('click', () => (p10.textContent = (+p10.textContent) ** 2));

Задача

Дан абзац с числом и кнопка. По нажатию на кнопку прибавьте к значению абзаца единицу и запишите полученное число обратно.

1

Решение:


      let p11 = document.querySelector('.p11');
      let button7 = document.querySelector('.button7');
      button7.addEventListener('click', () => (p11.textContent = +p11.textContent + 1));

Задача

Дан абзац с текстом и кнопка. По нажатию на кнопку запишите в конец текста абзаца восклицательный знак.

Ну, привеееет

Решение:


      let p12 = document.querySelector('.p12');
      let button8 = document.querySelector('.button8');
      button8.addEventListener('click', () => p12.textContent = p12.textContent + '!');

Задача

Модифицируйте предыдущую задачу так, чтобы восклицательный знак записывался не в конец текста, а в начало.

Ну, привеееет

Решение:


      let p13 = document.querySelector('.p13');
      let button9 = document.querySelector('.button9');
      button9.addEventListener('click', () => (p13.textContent = '!' + p13.textContent));
